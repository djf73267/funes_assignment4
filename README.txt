README

Binary Search Tree is a node based data structure that relies on the idea of having a single node as a central node and all nodes that derive from it are its children. Nodes without children are considered leaf nodes. Please read the Use section to identify how to correctly run this program

USE
use the makefile with the following command to create a excecution file
1. make main

to run the program with the given .txt file
2. ./main input.txt 

OTHER COMMANDS
make clean removes the ./main execution
